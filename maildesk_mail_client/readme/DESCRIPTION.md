MailDesk – Email Client & Unified Inbox for Odoo A modern, multi-account
email client for Gmail, Outlook, and any IMAP-compatible server – fully
integrated into Odoo.

> - Unified inbox with support for multiple accounts
> - OAuth2 integration for Gmail and Outlook
> - IMAP login support
> - Full-featured HTML email composer with inline images and attachments
> - Floating and resizable OWL-based composer interface
> - Auto-save and restore drafts seamlessly
> - Support for mail folders: Inbox, Sent, Trash, Archive...
> - Threaded conversation view
> - Shared inbox access for users
> - Real-time UX with background sync via scheduled jobs
> - Clear separation of access rights (user/admin)
> - GDPR-compliant backend, enterprise-ready architecture
