MailDesk is a modern, high-performance email client fully integrated into Odoo.  
It brings a unified inbox, multi-account support, a fast OWL interface, and seamless
integration with Gmail, Outlook, and any IMAP-compatible server.

MailDesk Basic is the core version of the MailDesk email client, providing the essential
functionality needed for efficient daily communication inside Odoo.

For the full 2-Way Sync experience, AI features, premium UX, and enterprise extensions,
please see **MailDesk Ultimate** (links below).

----------------------------------
Key Features (MailDesk Basic)
----------------------------------

- Unified inbox for multiple email accounts  
- Full HTML email composer with inline images & attachments  
- OAuth2 authentication for Gmail and Outlook  
- IMAP login support  
- Folder management (Inbox, Sent, Trash, Archive, etc.)  
- Threaded email conversations  
- Shared inbox capabilities  
- Auto-save drafts and recovery  
- Fast, reactive OWL-based interface  
- Background syncing (scheduled jobs)  
- GDPR-compliant backend  
- Clear user/admin access control  
- Modular architecture ready for upgrades to MailDesk Ultimate

----------------------------------
Required Python Libraries
----------------------------------
MailDesk uses several official and widely adopted libraries:

- ``imapclient``  
- ``aioimaplib``  
- ``google-api-python-client``  
- ``google-auth``  
- ``google-auth-httplib2``  
- ``msal``  

Install example (Ubuntu/Debian):

.. code-block:: bash

   pip3 install imapclient aioimaplib google-api-python-client google-auth google-auth-httplib2 msal

----------------------------------
Installation
----------------------------------

1. Install required Python libraries (see above).  
2. Deploy the module into your Odoo addons directory.  
3. Restart the Odoo server.  
4. Activate developer mode and update the module list.  
5. Install *MailDesk Basic* from the Apps menu.
