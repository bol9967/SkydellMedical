Start Where You Belong — Omux Home Menu

The Start Menu displays all available apps in a full-screen layout by default. Once a user pins any app, the view updates automatically: pinned apps are prioritised, and the sidebar reflects the same order.
Administrators can optionally predefine a default pinned list per company. This behaviour ensures a consistent user experience while allowing individual preferences to form naturally.

Highlights

  • ▸ Auto Switch to Favourites. Triggered when apps are pinned.
  • ▸ Workspace Palette. Organises views, shortcuts, and history.
  • ▸ Drawer Layout Toggle. Choose between fullscreen or standard mode.
  • ▸ Dynamic Sidebar Entries. Auto-update with your pinned favourites.
  • ▸ Per-Company Background. Includes blur and logo styling options.
  • ▸ Symbolic Icon Support. Keeps navigation clean and consistent.
  • ▸ Nested Menu Icon Support. Enables icons for nested menu items.

Frequently Asked Questions

If the answer to your question is not on this page, please contact our customer Support Team

How does the purchasing process work?
  • How to purchase? Add the module to your cart, proceed with payment, and wait for Odoo to validate your transaction. Then you'll be able to download the ZIP file.
  • Can I try before buying? Yes. We offer private demos on request — just make sure the feature descriptions match your expectations before testing.
  • Can I return after purchase? Returns are restricted. Please review carefully before buying. Our product quality is solid, and we fully support real issues. See Sales Conditions.
How to install or upgrade a module?

↳ Extract the ZIP to your addons path. Restart Odoo, update the Apps List, then install the module.

↳ When upgrading, make sure to upgrade all dependent modules from the ZIP if included.

↳ Some modules may require additional libraries — please check the product page for notes.

Does this module support multilingual language?

↳ Yes. Built following Odoo's i18n standards. Currently available in:

  • Arabic
  • English
  • Japanese
  • Simplified Chinese
  • Spanish
  • Turkish
  • Ukrainian
  • Vietnamese

↳ If your language is not listed, we’re happy to localize or improve existing translations.

How are recently viewed records managed?

↳ Only accessed form view records are tracked. They're saved when the tab or browser is closed.

↳ Duplicates are removed automatically to keep the list clean and efficient.

How to create similar records faster?

↳ Yes. Extended Creation Mode lets the “New” button duplicate your last record — perfect for batch creation.

↳ Useful for quotes, contacts, or items with repeated fields. Just edit what's new and save.

Does this theme support RTL layout?

↳ Yes, with full support — layout, side switching, and interactions all respect RTL.

↳ For Odoo Community users: ensure RTLCSS is installed. Then regenerate assets and switch language to RTL (e.g. Arabic). See Odoo docs.

Will Omux slow down my backend?

↳ Not at all. Assets are lazy-loaded and structured for performance. Omux stays lightweight even with full configuration enabled.

Can I request UI tweaks or feature extensions?

↳ Absolutely. We welcome new ideas and actively expand the Omux add-on ecosystem. Let's build what fits your workflow best.

Is there a license limit or restriction?

↳ Omux modules are licensed under OPL-1 (Odoo Proprietary License v1). Please follow Odoo's official licensing guidelines for production usage.

Do you offer support after purchase?

↳ Yes. We provide prompt support following the official Odoo Store policy. Reach out if you need help or run into issues.
