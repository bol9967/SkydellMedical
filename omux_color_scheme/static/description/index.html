Frequently Asked Questions

If the answer to your question is not on this page, please contact our customer Support Team

Do I need technical skills to use this module?

Basic technical knowledge is recommended. While the module works out-of-the-box, technical users can further customize SCSS parameters for advanced personalization.

Can I customize the preset color schemes?

Yes. The module comes with 8 built-in presets (Metronic, Slate, Haze, Gray, Neutral, Sage, Zinc, Mauve). Advanced users can modify or extend these presets by tweaking SCSS parameters as needed.

Will the module conflict with other themes or modules?

No. The design is non-intrusive and fully compatible with Odoo’s dark/light modes and custom themes. Its lazy-loaded components help ensure smooth integration.

How do I uninstall the module safely?

The module includes an uninstall hook that cleans up dynamically created assets, ensuring no residual files remain after uninstallation.

What future-proof advantages does it offer?

It leverages a forward-compatible design that seamlessly integrates with future Odoo updates, reducing maintenance and ensuring long-term reliability.

What performance benefits does it offer?

By employing dynamic asset injection and lazy-loaded components, the module reduces unnecessary load, resulting in faster initial load times and overall system efficiency.

How does dynamic asset injection work?

The module loads SCSS assets on demand, meaning only the required files are injected into the system. This minimizes overhead and boosts performance.
